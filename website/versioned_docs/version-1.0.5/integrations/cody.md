# Sourcegraph Cody Integration

Use Beads with Sourcegraph Cody through a project rules file.

```bash
bd setup cody
bd setup cody --check
```

The setup command creates `.cody/rules/beads.md` with Beads workflow guidance.

## Remove

```bash
bd setup cody --remove
```
