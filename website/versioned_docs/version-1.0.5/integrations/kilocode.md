# Kilo Code Integration

Use Beads with Kilo Code through a project rules file.

```bash
bd setup kilocode
bd setup kilocode --check
```

The setup command creates `.kilocode/rules/beads.md` with Beads workflow guidance.

## Remove

```bash
bd setup kilocode --remove
```
